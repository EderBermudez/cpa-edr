Proyecto EDR+Astro — carpeta `dist`

Este repositorio contiene los archivos estáticos del proyecto (HTML, CSS, imágenes y videos).

Pasos para subir a Bitbucket:

1. Crear un repositorio en Bitbucket (elige HTTPS o SSH).
2. En tu máquina local, configura el remoto y sube:

   git remote add origin <URL-DEL-REPO>
   git branch -M main
   git push -u origin main

Si usas HTTPS, crea un "App password" en Bitbucket para autenticación (recomendado sobre usar tu contraseña). Si prefieres SSH, añade tu clave pública a Bitbucket.
